วิธีติดตั้งบน Android

ใช้ Chrome, Edge, หรือ Firefox

  1. เปิด URL ของแอป บน Chrome หรือ Edge
  2. รอให้เห็นป็อปอัพ "ติดตั้ง" ที่มุมล่าง (หรือกดเมนู ⋮ → Install)
  3. แตะปุ่ม "ติดตั้ง" เพื่อยืนยัน
  4. เสร็จแล้ว! แอปจะปรากฏบนหน้าจอหลัก

วิธีติดตั้งบน iOS (iPhone/iPad)

ใช้ Safari เท่านั้น

  1. เปิด Safari บน iPhone หรือ iPad
  2. ไปยัง URL ของแอป พิมพ์ที่แถบที่อยู่
  3. แตะปุ่มแชร์ 🔄 ที่ด้านล่าง (หรือทั้งสองชั้น)
  4. เลือก "Add to Home Screen"
  5. ตั้งชื่อ "ThaiBank" แล้วแตะ Add

วิธีติดตั้งบน PC/Mac

ใช้ Chrome, Edge, หรือ Brave

  1. เปิด URL ของแอป ใน Chrome หรือ Edge
  2. ไอคอนติดตั้งจะแสดง ที่ขึ้นแถบที่อยู่
  3. แตะไอคอน หรือใช้เมนู ⋮ → "Install ThaiBank"
  4. เปิดในหน้าต่างแยก พร้อม UI เต็ม
